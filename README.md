📊 DataLens
DataLens is a full-stack web application for managing and visualizing data with a clean UI and a fast backend. It’s built to demonstrate practical full-stack development skills including authentication, CRUD operations, and deployment.

🔗 Live Demo
👉 Project Link: https://datalens.kwscloud.in/

🚀 Features
📦 Create, view, update, and delete data entries

🔐 Secure login system

👨‍💼 Admin access for full control

📊 Responsive and clean user interface

⚡ REST API with fast backend performance

🗄️ SQLite database integration

🛠️ Tech Stack
Frontend
React.js

HTML, CSS, JavaScript

Backend
Node.js

Express.js

Database
SQLite

Deployment
KWS - Kamaraj Web Service


⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/Joel-93/DataLens.git
cd DataLens


2. Install dependencies
npm install

3. Run backend

cd backend
node server.js


4. Run frontend

npm start




⚠️ Important Notes

Avoid using relative API paths like /api in production

SQLite is used for simplicity and demo purposes
